Sobre o Site

Do que se Trata?

Essa página foi criada utilizando os conhecimentos obtidos no primeiro módulo do curso de HTML5 da DIO, servindo como um meio de exercitar e fixar o que foi aprendido.

O que foi Apresentado?

A página tinha como base um desafio proposto no fim do primeiro módulo, onde deveríamos apresentar as tags aprendidas e suas respectivas funções dentro do HTML. Simples e breve, mas sendo uma ótima prática de fixação, apliquei um pouco de CSS (estudado por fora) para incrementar um visual mais agradável nos blocos de citação, mas sem aprofundar demais para não fugir do contexto geral.
