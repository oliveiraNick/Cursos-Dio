Sobre o HTML

Primeiramente, devemos deixar claro que o HTML não é uma linguagem de programação, é uma ferramenta utilizada no desenvolvimento web para definir a estrutura da página, sem poder criar condições, interatividade ou um conteúdo dinâmico (isso sem o uso de outras feramentas), chamada de Linguagem de Marcação.
Isso, claro, podemos encontrar na internet da seguinte forma:

o HTML é como o esqueleto de uma página web, organizando os elementos como texto, imagens, vídeos e links.

Como o HTML Funciona?

A estrutura geral de uma página é formada pelo HTML, um arquivo .html mais especificamente. Para cada página no nosso site temos um arquivo desses como estrutura básica.
Dentro dele temos uma sequência de tags que dão sentido ao texto, elas permitem que o navegador consiga ler, interpretar e apresentar aquela página corretamente para o usuário. De forma simples:

As tags em HTML, também chamadas de elementos, são instruções que definem a estrutura e o significado do conteúdo em uma página web. Elas são como blocos de montar que organizam e formatam os elementos da página, como texto, imagens, vídeos, links, formulários e outros.

Tags da Estrutura Básica

Fazem parte da estrutura básica de todo arquivo HTML, alguams delas são:

  • <!DOCTYPE html>: Tag fundamental para a construção da página web, ela diz ao navegador que se trata de um arquivo .html permitindo uma melhor interpretação do mesmo;
  • <html>: Define o início do HTML;
  • <head>: Dentro dessa tag encontramos as configurações do arquivo, como títulos e metadados;
  • <body>: O corpo da nossa página, nessa tag iremos colocar o conteúdo que irá aparecer para o usuário;

Tags de Formatação

Controlam questões visuais e de semântica do conteúdo, sendo algumas delas:

  • <p>: Define um texto como um parágrafo;
  • <strong>: Deixa um texto em negrito, dando destaque a ele;
  • <em>: Da ênfase ao texto, deixando ele no estilo itálico;
  • <br>: Faz uma quebra de linha em qualquer lugar do texto;
  • <u>: Destaca o texto sublinhando ele;
  • <h1> ao <h6>: Define a hierarquia do título, sendo 1 título printipal (h1) e 5 subtítulos (h2, h3, h4, h5 , h6).

Tags Semânticas

Definem o significado do conteúdo da página, sendo algumas delas:

  • <header>: Define o cabeçalho da página;
  • <footer>: Define o rodapé da página;
  • <article>: Define um texto como um artigo independente;
  • <main>: Define o conteúdo principal da página;

Tags de Conteúdo

Serve para adicionar outros tipo te conteúdo na página fora texto, sendo algumas delas:

  • <a>: Utiliza-se quando deseja adicionar um link externo clicável a página.
  • <img>: Como o nome mesmo diz, é utilizado para adicionar imagens ao conteúdo;
  • <ul>: Utilizada quando queremos adicionar uma lista não ordenada ao conteúdo;
  • <table>: Essa tag é usada quando se deseja adicionar uma tabela ao conteúdo.
